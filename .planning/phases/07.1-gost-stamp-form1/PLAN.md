### 5. Боковые поля и маркировка формата — `drawSheet` ([backends/pdf.ts:91-111](../../src/export/backends/pdf.ts))

Боковая полоса слева (между обрезом листа и левой границей рамки):
- Ширина ~7 мм (FRAME_MARGIN_LEFT_MM=20, можно использовать 5..18 от обреза)
- Графы (повёрнутый текст 90° против ч.с.):
  - «Согласовано» — верх
  - «Инв. № подл.» — пустая строка
  - «Подп. и дата» — пустая строка  
  - «Взам. инв. №» — пустая строка
  - Каждое поле — 25-40 мм по высоте, разделены тонкой линией

Маркировка формата:
- Под рамкой справа внизу (за пределами рамки)
- Текст `Формат ${formatCode}${orientCode}`, например `Формат А4К`, `Формат А3А`, `Формат А4×5А`
- `formatCode` — из `format.label` без префикса (`А4`, `А3`, `А4×5`)
- `orientCode` — `К` для portrait, `А` для landscape
- Шрифт мелкий (5-6 pt), серый

### 6. Маркер К/А

В [sheet/formats.ts](../../src/export/sheet/formats.ts) добавить функцию:
```ts
export function formatStampMark(format: SheetFormat, orientation: Orientation): string {
  const baseLabel = format.label  // "А4", "А3×3"
  const orient = effectiveOrientation(format, orientation)
  const mark = orient === 'portrait' ? 'К' : 'А'
  return `Формат ${baseLabel}${mark}`
}
```

### 7. Режимы листа `stampMode`

В `drawSheet` (PDF) и `SheetCanvas` (превью):
- `'full'` (default) — рисует штамп + боковую полосу + маркировку формата
- `'minimal-footer'` — без штампа и боковой полосы; внизу справа за рамкой строка `footerLine` (типа «Приложение Б»). Маркировка формата остаётся.
- `'none'` — только рамка + контент. Ни штампа, ни боковой полосы, ни маркировки.

`paginate()` ([layout.ts](../../src/export/sheet/layout.ts)) — `usableHeightMm` зависит от режима: для `'full'` минусуем штамп, для остальных только небольшой нижний отступ.
